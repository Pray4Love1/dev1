Tempo API method that returns the receipt of a transaction by transaction hash. Transaction receipts contain information about the execution of a transaction.

<Info>
**Tempo-specific fields**: Transaction receipts on Tempo include additional fields:
- `feeToken` ‚Äî the TIP-20 token address used to pay transaction fees
- `feePayer` ‚Äî the address that paid the fees (may differ from sender with fee sponsorship)
</Info>


## Parameters

* `transactionHash` ‚Äî the hash of the transaction to retrieve the receipt for

## Response

* `result` ‚Äî a receipt object, or `null` if no receipt was found:
  * `transactionHash` ‚Äî the transaction hash
  * `blockHash` ‚Äî hash of the block containing this transaction
  * `blockNumber` ‚Äî block number containing this transaction
  * `from` ‚Äî address of the sender
  * `to` ‚Äî address of the receiver
  * `status` ‚Äî `1` for success, `0` for failure
  * `gasUsed` ‚Äî amount of gas used
  * `logs` ‚Äî array of log objects
  * `feeToken` ‚Äî TIP-20 token address used for fees (Tempo-specific)
  * `feePayer` ‚Äî address that paid the fees (Tempo-specific)
