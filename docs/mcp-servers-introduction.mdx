---
title: "MCP servers"
description: "Connect AI agents to the Chainstack platform via the Model Context Protocol"
---

Chainstack provides a unified [MCP (Model Context Protocol)](https://modelcontextprotocol.io/) server at `https://mcp.chainstack.com/mcp` that gives AI agents access to the entire Chainstack platform.

Type this into any AI agent to get started:

```
get mcp.chainstack.com
```

See [Chainstack MCP server](/docs/chainstack-mcp-server) for the full setup guide, tools, and per-agent configurations.
